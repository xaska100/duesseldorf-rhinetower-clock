FernsehturmDdorfClock
=============
Java class that draws the Düsseldorf Rhine Tower and displays *your local time* on it.
The background color varies depending on the *current time in Düsseldorf*.


This project is using Java utilities StdDraw (https://introcs.cs.princeton.edu/java/stdlib/javadoc/StdDraw.html) and SunriseSunset (https://github.com/caarmen/SunriseSunset).
